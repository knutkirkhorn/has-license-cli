# has-license-cli
> Check if a repository has a license CLI

## Installation
```
$ npm install --global has-license-cli
```

## Usage
```
$ has-license --help

        Usage
          $ has-license <path>

        Examples
          $ has-license
          $ has-license https://github.com/knutkirkhorn/has-license
          $ has-license C:\Users\Knut\Documents\GitHub\has-license 
```
